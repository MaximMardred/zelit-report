Настройки Supabase

Подключите облачную базу данных для совместной работы команды

Как подключить Supabase — пошаговая инструкция

Supabase — это бесплатная облачная база данных. После подключения все отчёты будут храниться на сервере и доступны для всей команды из любого браузера.

1
Перейди на supabase.com и зарегистрируйся (бесплатно, через GitHub или почту)
2
Нажми New Project, придумай название (например reportflow), выбери регион Central EU или West EU, придумай пароль базы данных (сохрани его!)
3
Подожди ~2 минуты пока проект создаётся. Потом в левом меню нажми SQL Editor и вставь этот SQL, нажми Run:

create table reports (id uuid primary key default gen_random_uuid(), client text not null, period text not null, data jsonb not null, created_at timestamptz default now(), updated_at timestamptz default now(), unique(client, period));
4
В левом меню нажми Project Settings → API. Скопируй Project URL и anon public key — вставь их в поля ниже
5
Нажми Сохранить настройки ниже. Готово — теперь все отчёты хранятся в облаке и доступны всей команде!
